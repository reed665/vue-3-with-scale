# vue-3-with-scale

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Run built app from dist folder
```
yarn preview
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
